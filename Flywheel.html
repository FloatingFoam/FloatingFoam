Flywheel Nerf guns work by using a pair of spinning flywheels powered by electric motors to shoot darts. When the trigger is pulled, a dart pusher moves forward, pushing the dart into the spinning flywheels, which grip the dart and propel it through the air. (smartnerf.com is where I got the flyweel information bc i didnt know how to reword that)


How do I tell a flywheel blaster from any other type of blaster?

There is a little button underneath used to start the flywheels, which easily gives it away. In older designs like the Prometheus, you can easily see the flywheels. Most nerf blasters that are motorized use flywheels, so it is pretty easy to find them.
